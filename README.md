# Проєкт: WebStudio (Домашня робота №3)

Цей репозиторій містить третю домашню роботу. Основна мета цього модуля — освоєння блокової моделі, налаштування геометрії елементів та позиціонування контенту за допомогою **Flexbox**.

## 🛠️ Що було зроблено в цьому модулі:

- **Глобальне скидання стилів:** Налаштовано базові "нульові" відступи для заголовків `<h1>-<h6>`, абзаців `<p>` та списків `<ul>` для точного контролю геометрії.
- **Створення сітки (Flexbox):** \* Елементи хедера (логотип, навігація, контакти) вибудувані в один рядок з вирівнюванням по вертикалі.
  - Картки в секціях **Our Features** та **Our Team** розташовані горизонтально з фіксованою шириною карток (`264px`) та проміжками `gap: 24px`.
  - Секція **Our Portfolio** реалізована у вигляді адаптивної сітки з переносом рядків (`flex-wrap: wrap`), де кожна картка має ширину `360px`.
- **Обмежувальний Контейнер:** Створено універсальний клас `.container` з фіксованою шириною `1158px` (1128px контенту + по 15px паддінгів з боків) для ідеального центрування сайту на великих екранах.
- **Прихований заголовок:** Заголовок секції переваг приховано з екрана за допомогою доступного для роботів та скрінрідерів патерну `.sr-only`.
- **Оформлення карток:** Для карток портфоліо додано рамку (`border`) з трьох боків (знизу, зліва та справа) на стику з зображенням за макетом.
